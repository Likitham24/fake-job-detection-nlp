# Fake Job Detection using NLP

This project detects fraudulent job postings using Natural Language Processing and Machine Learning.

## Live App
https://likitha-fake-job-app.streamlit.app

## Technologies Used
- Python
- Pandas
- Scikit-learn
- NLP
- Streamlit

## Model
Logistic Regression with TF-IDF Vectorization

## Dataset
Fake Job Postings Dataset
